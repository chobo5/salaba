### 2. 기능 계획
1. 웹툰
   - 웹툰 추천 기능
   - 요일, 장르별, 완결, 신작, 무료 카테고리별 분류
2. 도서
   - 도서 추천 기능
   - 베스트, 스테디, 신작 카테고리별 분류
3. 웹툰 그려보기 기능 제공
   - 일반 유저가 직접 웹툰을 그려볼 수 있는 기능 제공

4. 게시판
  - 게시판은 주제를 특정 웹툰으로 선정 가능
  - 글쓰기에 그려보기 기능 연동
